Overview

CODELOGY - Learning Programming Mobile App

The Problem

There are 40,000 IT graduates produced annually, and only 1% successfully secure positions in large companies and can compete in the global arena. Many IT students are found to be unable to code due to a lack of learning guidance and difficulties in understanding the material.

My Role

My role in the success of this project as a UI/UX Designer is to execute the design process using the design thinking method, create wireframes along with high-fidelity designs, and develop prototypes. Following that, I will commence testing using the User Experience Question (UEQ) form distribution method.
